# flightwith-vue
An informational website that alerts its users about cheap airline tickets.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Run your unit tests
```
npm run test:unit
```
## General info
We use:
* Vue - JavaScript Framework
* Firebase Hosting
